# Movie Library

**Автор:** Игнатова Варвара

## Описание
Приложение для хранения личной коллекции фильмов.  
Позволяет добавлять, удалять, фильтровать фильмы по жанру и году выпуска.  
Все данные сохраняются в файл `movies.json`.

## Установка и запуск
1. Убедись, что установлен Python 3.6+
2. Скачай репозиторий
3. Запусти:
   ```bash
   python movie_library.py
